## What is Clayton Forge?

Clayton Forge started as a **comprehensive methodology** for building AI agents — 39 reference documents covering everything from cognitive architecture to production runbooks.

It's now evolving into a **complete platform**: CLI, MCP Server, and Web UI — all powered by the same methodology underneath.

> Build agents the right way. Ship them with confidence.

## ⚡ Quick Start

### CLI (Fase 1 — available now)

```bash
# Clone and install
git clone https://github.com/devclaytonn10/clayton-forge
cd clayton-forge
npm install
npm run build --workspaces

# Create your first agent
node packages/cli/dist/index.js new

# Or when published to npm:
npm install -g clayton-forge
cforge new
```

### Commands

```bash
cforge new        # Interactive wizard — create a new agent
cforge run        # Execute the agent in the current directory
cforge test       # Run the test suite
cforge validate   # Validate agent.yaml without executing
```

### Your agent in 60 seconds

```yaml
# agent.yaml (generated by cforge new)
name: "invoice-classifier"
version: "1.0.0"
type: reactive
trust_level: 1

llm:
  provider: anthropic          # or openai | ollama | lmstudio | none
  model: claude-sonnet-4-6
  api_key: "${ANTHROPIC_API_KEY}"

prompt:
  system: "Classify invoices by category and urgency. Respond in JSON."
  user_template: "{{input}}"
  variables:
    - name: input
      required: true

output:
  format: json
```

```bash
cforge run --input "Electricity bill $245 due in 3 days"
# → { "category": "utilities", "urgency": "high" }
```
